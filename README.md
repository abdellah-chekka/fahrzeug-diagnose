# Fahrzeug-Fehlerdiagnose

Erstellt von Abdellah Chekka am 06. März 2026

## Über das Projekt

Ein Python-Tool, das Fahrzeugsensoren simuliert und automatisch auf Fehler prüft. Entwickelt für meine Bewerbung für die Ausbildungsstelle als Fachinformatiker für Anwendungsentwicklung bei VW Infotainment.

## Funktionen

- Simuliert Sensordaten (Geschwindigkeit, Motortemperatur, Reifendruck, Batterie)
- Prüft automatisch ob Werte im Normalbereich liegen
- Erkennt Fehler wie Überhitzung oder niedrigen Reifendruck
- Speichert Fehlerberichte als JSON-Datei
